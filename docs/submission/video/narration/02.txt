PPP is not a code generator followed by file-save hot reload. It is an AI-era product REPL: evaluate behavior in long-lived server and browser runtimes, observe the real result, repair it, and checkpoint what works. This shared public proof uses a bounded staged runtime. The destination is one isolated Workspace Capsule per project, where Codex can use nREPL to redefine the running Clojure server directly.
