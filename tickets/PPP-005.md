# Outcome

Generated shared and server source stages in a fresh capability-limited SCI context against a consistent copy of the session database. Valid actions can query and mutate user tables; evaluator, SQL, and kernel-table escapes fail closed.

# Scope

- Single capability catalog and generated documentation view.
- Server SCI context with copied namespaces/symbols and interrupt deadline.
- Deterministic shared-before-server evaluation order.
- Staged action registry and host contract validation.
- SQLite initialization, foreign keys, busy timeout, runtime metadata.
- Host-sequenced immutable migrations and migration hashes.
- Read/write action wrappers with parameter binding.
- Logical user-data hash and consistent staging snapshot.
- Demo gallery domain, actions, migration, seeds, score, and tie behavior through fake provider source.

# Acceptance Criteria

- [x] Server SCI has no general Java class, IO, shell, process, thread, eval, loader, or filesystem access.
- [x] Infinite or excessive generated evaluation terminates by deadline.
- [x] Required server entrypoint stages or yields a bounded diagnostic.
- [x] Active registry is not replaced during staging.
- [x] Generated migration cannot mention `_ppp_`, ATTACH, PRAGMA, extension, filesystem, temporary schema, or edit a committed migration.
- [x] Action SQL is one parameterized SELECT/INSERT/UPDATE/DELETE statement.
- [x] Stage DB failure leaves live DB logical hash unchanged.
- [x] Generated domain tests execute against staged actions before commit and their writes always roll back.
- [x] Missing, empty, or failed generated tests reject staging with bounded repair feedback and cannot call outbound HTTP/connectors.
- [x] UI-only changes and committed-runtime startup do not replay domain tests against mutable live data; generated tests use fixtures or baseline deltas.
- [x] Six gallery projects seed once and persist.
- [x] Judge/public scoring and deterministic ties match domain rules.

# Domain/PBT Tests

- Generate forbidden symbols/forms and prove no host capability is reached.
- Generate allowed/forbidden SQL statements, casing, whitespace, comments, and quoted strings.
- PBT-07: generated vote sequences match the pure domain scorer and tie order.
- Migration replay is idempotent; same name with different hash fails.
- Do not test exact SQL formatting or private function call order.
