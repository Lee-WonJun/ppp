# Outcome

Generated server actions can call explicitly bounded HTTPS public endpoints and developer-owned connector aliases without reaching private infrastructure or revealing secret headers to the model or generated runtime.

# Scope

- URL parser and normalized policy.
- Public/reserved IPv4 and IPv6 classifier.
- Kernel-owned DNS resolution and address pinning.
- TLS hostname validation with pinned address connection.
- Redirect revalidation and limits.
- Method/header/body/timeout/response-size limits.
- `connectors.edn` loader and schema.
- Alias/path/method policy and host-side secret injection.
- Response normalization and bounded diagnostic codes.

# Acceptance Criteria

- [x] Public capability accepts HTTPS only.
- [x] Any private, loopback, link-local, multicast, metadata, documentation, or reserved DNS result fails closed.
- [x] Mixed public/private DNS results fail closed.
- [x] Every redirect repeats URL and DNS validation.
- [x] Arbitrary Authorization, Cookie, forwarding, proxy, host, and hop-by-hop headers are rejected.
- [x] Response is limited to 1 MiB after decompression and five-second default timeout.
- [x] Connector model context contains no secret value or env name.
- [x] Secret injection happens only after alias, method, path, and target pass policy.
- [x] Logs and action errors contain no injected header.

# Domain/PBT Tests

- PBT-06 generates URLs, IPs, DNS sets, and redirect chains across boundary ranges.
- Local controlled HTTPS server proves public success, redirect checks, timeout, decompression limit, and header filtering.
- Connector fixtures prove prefix normalization cannot escape with encoding or dot segments.
- Captured provider context and logs prove redaction.

# Evidence

- address-range property report
- controlled HTTP integration trace
- redacted connector capability catalog
- negative request matrix

Recorded in `artifacts/evidence/ppp-009-outbound.md`.

# Out of Scope

- User-created connectors, OAuth consent, secret UI, or arbitrary custom headers.
- Browser-side public fetch.
- Non-HTTPS protocols.
